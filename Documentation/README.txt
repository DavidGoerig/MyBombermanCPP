INDIE STUDIO:
Bomberman like project made in C++ with Irrlicht lib

Building:
cmake -G "Unix Makefiles"
./bomberman

Documentation:
You will find all the documentation in the ./Documentation folder.
Create with Doxygen.

UML:
You will find the UML in the ./ULM folder.
Create with Visual Paradigm.
